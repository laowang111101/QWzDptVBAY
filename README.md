# 重点人员信息管理系统 python802

## 项目概述

重点人员信息管理系统是一款基于Python3.6.5、pyqt5和MySQL开发的管理软件。系统主要用于对重点人员信息进行高效管理，包含用户管理、管控类别管理和辖区人员管理等模块。

## 技术栈

- Python 3.6.5
- PyQt5
- MySQL

## 功能模块

### 用户管理

- 管理员登录：使用`admin`账号作为管理员
- 用户信息管理：实现对用户信息的增删改查操作

### 管控类别管理

- 数据操作：包括增删改查功能

### 辖区人员管理

- 数据操作：包括增删改查功能

### 重点人员信息管理

- 数据操作：包括增删查功能

## 系统角色

- 管理员：负责系统维护及用户管理

## 运行环境

- 操作系统：兼容Windows、Linux等常见操作系统
- Python版本：Python 3.6.5及以上
- 数据库：MySQL

## 部署步骤

1. 安装Python环境（3.6.5版本）
2. 安装PyQt5库
3. 安装MySQL数据库，并执行sql脚本建立数据库结构
4. 部署源代码
5. 运行程序
